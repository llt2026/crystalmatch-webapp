## 🔧 如何获取PayPal专用测试账号

### 步骤1：访问PayPal Developer Dashboard
```
https://developer.paypal.com/developer/accounts/
```

### 步骤2：创建沙箱账号
1. 点击 "Create Account"
2. 选择 "Personal" (买家账号)
3. 选择国家
4. 点击 "Create Account"

### 步骤3：获取测试凭据
- 邮箱地址：自动生成
- 密码：可设置
- 余额：可配置
